# HH Greece project

## Project description
This project is to practise my skills in HTML and CSS. The aim is to create a simple website for a small travel agency.

## Structure of the project
- <code>index.html</code> - the main HTML file containing the structure of the website
- <code>style.css</code> - CSS file for styling the website
- <code>lightbox-plus-jquery.min.js</code> - JavaScript file containing code for gallery
- img: directory containing images for this website
